This repository contains:
    MCBPO.pdf: the written report
    MCBPO.owl: the owl ontology
    MCBPO_data.py: the python script to obtain the data from the UniProt databases
    MCBPO_queries.py: the python script to do the sparql queries
    uniprot_sample.csv: file containing the information introduced in our ontology
    instances_files.R: R script that extracts from uniprot_sample.csv the necessary information for the bash scripts
    function_script.txt, gene_script.txt, location_script.txt, prot_script.txt and species_script.txt: the bash scripts to generate the RDF version of uniprot_sample.csv
    Documentation: the documentation obtained with Widoco
